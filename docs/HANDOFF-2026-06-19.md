### 1-A. `frontend/src/pages/Deals.tsx`
表本体（`<div className="table-scroll">` 内）の `<table>`〜`<tbody>` の map を以下の最終形にする。
ポイント: ① `<table>` に `deals-grid` を追加、② `COLS.map((c) => …)` を `COLS.map((c, i) => …)` にして、③ 先頭2列（`i < 2`）に `stickL stickL${i}` クラスを付与。

```tsx
            <table className="sticky-head deals-grid">
              <thead>
                <tr>
                  {COLS.map((c, i) => (
                    <th key={c.key}
                      className={[c.align === "num" ? "num" : "", i < 2 ? `stickL stickL${i}` : ""].filter(Boolean).join(" ") || undefined}
                      style={{ textAlign: c.align === "center" ? "center" : undefined, cursor: c.sort ? "pointer" : "default", userSelect: "none", whiteSpace: "nowrap" }}
                      onClick={() => c.sort && toggleSort(c.key)}>
                      {c.label}
                      {c.sort && <span style={{ fontSize: 9, opacity: 0.55, marginLeft: 3 }}>
                        {sort?.key === c.key ? (sort.dir === "asc" ? "▲" : "▼") : "⇅"}</span>}
                    </th>
                  ))}
                  <th></th>
                </tr>
              </thead>
              <tbody>
                {display.map((d) => (
                  <tr key={d.id}>
                    {COLS.map((c, i) => (
                      <td key={c.key}
                        className={[c.align === "num" ? "num" : "", i < 2 ? `stickL stickL${i}` : ""].filter(Boolean).join(" ") || undefined}
                        style={{ textAlign: c.align === "center" ? "center" : undefined }}>
                        {c.render(d, ctx)}
                      </td>
                    ))}
                    <td style={{ display: "flex", gap: 6 }}>
                      <button className="btn sub sm" onClick={() => navigate(`/deals/${d.id}/edit`)}>編集</button>
                      <button className="btn sub sm" onClick={() => remove(d.id)}>削除</button>
                    </td>
                  </tr>
                ))}
              </tbody>
            </table>
```

### 1-B. `frontend/src/styles.css`
末尾（または `.sticky-head` 定義の近く）に追記。
`.sticky-head thead th` は既に「上方向 sticky・背景・z-index:1」が付いているので、左固定を足す。

```css
/* 案件一覧：左2列（実施予定・顧客名）を横スクロール時に固定 */
.deals-grid .stickL { position: sticky; background: var(--panel); z-index: 1; }
.deals-grid thead .stickL { z-index: 3; }          /* 見出しの固定列は最前面 */
.deals-grid .stickL0 { left: 0; width: 76px; min-width: 76px; }
.deals-grid .stickL1 { left: 76px; box-shadow: 1px 0 0 var(--border); } /* 顧客名の右端に区切り線 */
```

### 1-C. 注意・微調整
- `stickL1` の `left: 76px` は **`実施予定` 列(stickL0)の幅(76px)と一致させる**こと。
  実機で「実施予定」列の見出し幅がもっと広い/狭い場合は、`stickL0` の `width` と `stickL1` の `left` を**同じ値**で調整する（例: 両方 70px / 84px など）。
- `position: sticky` は背景が透明だと下の列が透ける。`background: var(--panel)`(=白) を入れてあるのでOK。行ホバー背景は無いので問題なし。
- ヘッダーは「上固定 × 左固定」のコーナーになる。`thead .stickL { z-index: 3 }` で前面に出している。
